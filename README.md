## тестовое задание hammersystem

POST api/v1/login запрос на авторизацию пользователя (номер и пароль)
POST api/v1/verify запрос на подтверждение 4-значного кода и авторизации (номер и код)
GET api/v1/profile/<int:pk> запрос профиля конкретного пользователя 
POST api/v1/profile запрос на ввод инвайт кода
GET api/v1/logout выход из сессии авторизованного пользователя
